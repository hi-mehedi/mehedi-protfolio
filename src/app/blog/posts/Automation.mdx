## Overview

The **AccuWeather Automation Framework** is built using **Aquality Selenium (Java)** and **TestNG**, designed to test weather forecast functionalities and ensure platform reliability. It also integrates **Allure Reporting** for detailed test insights.

This framework demonstrates best practices in **automation architecture**, **test scalability**, and **reporting setup**.
