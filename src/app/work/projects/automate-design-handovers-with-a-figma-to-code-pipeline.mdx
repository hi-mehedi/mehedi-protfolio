## Overview

This project showcases a comprehensive test automation framework built with Aquality Selenium for robust web testing, integrated with Allure Report for beautiful and detailed test reporting. The framework demonstrates maintainable, scalable automation practices using Java, Maven, and TestNG.

## Key Features

- **Aquality Selenium Integration**: Leveraged Aquality's built-in capabilities for element waiting strategies, browser management, and enhanced selenium wrappers
- **Allure Report Integration**: Implemented detailed test reporting with screenshots, steps, and attachments for comprehensive test analysis
- **Page Object Model**: Designed a maintainable framework structure following best practices
- **Data-Driven Testing**: Implemented parameterized tests using TestNG data providers
- **CI/CD Ready**: Configured for seamless integration with Jenkins/GitHub Actions

## Technologies Used

- **Java**: Core programming language for test automation
- **Maven**: Dependency management and build automation
- **TestNG**: Test framework for parallel execution and advanced test configuration
- **Aquality Selenium**: Enhanced selenium wrapper for reliable automation
- **Allure Report**: Beautiful test reporting and dashboard
- **GitHub Actions**: CI/CD pipeline integration

## Challenges and Learnings

Implementing reliable wait strategies and handling dynamic content were key challenges. The integration of Allure reporting provided valuable insights into test execution patterns and failure analysis.

## Outcome

The framework significantly reduced regression testing time by 70% and provided comprehensive test reports that helped developers quickly identify and fix issues.
